# SignalDaily Source Map

Use this as the priority map for daily research. Prefer primary sources first, then reputable secondary reporting for context.

## AI and frontier models

| Source type | Examples | Use for |
|---|---|---|
| Primary labs | OpenAI, Anthropic, Google DeepMind, Meta AI, Microsoft AI, xAI | Model launches, safety reports, product updates, benchmarks, policy changes |
| Research | arXiv, Papers With Code, conference proceedings | Technical papers, evaluations, new methods |
| Developer docs | GitHub, Microsoft, Google Cloud, AWS, Cloudflare, Vercel | SDKs, APIs, agent tooling, platform changes |
| Trusted media | The Verge, TechCrunch, Bloomberg, Reuters, FT, AP | Market context, company strategy, executive comments |

## Coding and developer tools

| Source type | Examples | Use for |
|---|---|---|
| Official repos/docs | GitHub releases, project changelogs, package registries | Tool releases, migration notes, breaking changes |
| Security research | GitHub Security Lab, Wiz, Trail of Bits, Snyk, Socket, CISA | Vulnerabilities, supply-chain risk, agent-security issues |
| Open-source signal | GitHub Trending, Hacker News, Reddit programming communities | Early adoption signal; verify before treating as fact |

## Sri Lanka, CSE, and economy

| Source type | Examples | Use for |
|---|---|---|
| Official | CBSL, CSE, SEC Sri Lanka, Department of Census and Statistics, Treasury | Policy, rates, inflation, reserves, market statistics, official notices |
| Company primary | CSE company announcements, annual/interim reports | Dividends, rights issues, earnings, governance, disclosures |
| Local business media | EconomyNext, Daily FT, NewsWire, Lankadeepa business, Ada Derana Biz | Context and quick summaries; verify high-impact items against primary sources |
| International | World Bank, IMF, ADB, Reuters, AP | Macro context, country classification, debt, trade, global linkages |

## Global markets

| Source type | Examples | Use for |
|---|---|---|
| Official/economic data | Federal Reserve, ECB, BLS, BEA, EIA, OPEC, Treasury | Rates, inflation, employment, commodities, macro releases |
| Market data | Nasdaq, NYSE, CME, CoinDesk, exchange pages | Prices, closes, volumes, crypto/commodity references |
| Reputable reporting | Reuters, AP, Bloomberg, MarketWatch, FT, WSJ | Market interpretation and institutional context |

## Source discipline rules

- Do not rely on a single source for high-impact claims.
- Use exact dates for time-sensitive claims.
- Mark Sunday/holiday CSE data as last trading-session data when applicable.
- Separate facts from interpretation.
- Use direct URLs in the Sources section.
- Do not cite social posts as final proof unless they are from an official verified account and no better primary source exists.
- For financial topics, provide educational context only. Do not recommend buying, selling, holding, or trading.
