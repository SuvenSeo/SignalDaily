# Telegram Alert Template

Use this template for short Telegram bot/channel updates.

```text
SignalDaily — YYYY-MM-DD

Top signals:
1. [AI / model / tech signal]
2. [Coding / developer tools signal]
3. [Sri Lanka / CSE / economy signal]
4. [Global markets signal]
5. [Opportunity / project idea]

Full report:
daily/YYYY-MM-DD.md
```

## Rules

- Keep the alert under 1,000 characters.
- Do not include long analysis.
- Include only the top signals and file path.
- Use direct, clean wording.
- Do not include financial advice.
