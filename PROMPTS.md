# SignalDaily Prompts

Use these prompts with the SignalDaily ChatGPT Agent.

## Daily run

```text
Run today's SignalDaily brief. Make it GitHub-ready for SuvenSeo/SignalDaily. Include the suggested file path, commit message, and Telegram-ready alert message.
```

## GitHub-ready daily report only

```text
Create today's SignalDaily report as clean Markdown only. Use the repository format from AGENT.md. Save-path suggestion must be daily/YYYY-MM-DD.md.
```

## Opportunity radar only

```text
Give me only today's Opportunity Radar with 3 coding ideas, 2 startup/SaaS ideas, 1 Sri Lanka-specific opportunity, and 1 market-research idea. Keep it practical and explain why each idea matters now.
```

## Deep dive

```text
Pick the biggest AI, tech, CSE, or global market signal today and write a Deep Dive of the Day. Explain what happened, why it matters, who is affected, what to watch next, and how I can use this knowledge.
```

## Sri Lanka / CSE focus

```text
Run a Sri Lanka-focused SignalDaily brief covering CSE announcements, CBSL updates, LKR/USD movement, listed-company developments, local business news, and market education. Do not give buy/sell advice.
```

## AI and coding focus

```text
Run an AI and coding-focused SignalDaily brief covering model releases, AI agents, coding assistants, GitHub trending repos, IDEs, frameworks, and developer workflow shifts. Include project ideas I can build.
```

## Telegram alert generation

```text
Create a short Telegram-ready alert for today's SignalDaily report. Keep it under 1,000 characters. Include the top 5 signals and the GitHub file path.
```

## Weekly recap

```text
Create this week's SignalDaily recap. Summarize the biggest AI, tech, coding, Sri Lanka/CSE, global market, and opportunity trends from the daily reports. Save-path suggestion must be weekly/YYYY-WW.md.
```

## Monthly intelligence review

```text
Create a monthly SignalDaily review. Identify recurring themes, major shifts, missed signals, strong opportunities, and topics to monitor next month. Save-path suggestion must be monthly/YYYY-MM.md.
```

## Commit message format

Use this format for daily commits:

```text
daily: add SignalDaily brief for YYYY-MM-DD
```

Weekly:

```text
weekly: add SignalDaily recap for YYYY-WW
```

Monthly:

```text
monthly: add SignalDaily review for YYYY-MM
```
